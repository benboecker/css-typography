CSS Typographie
==============

Dies ist ein Beispielprojekt, welches einige Techniken vorstellt, um typographische Konzepte auf einer Webseite mittels CSS anzuwenden.

Folgende CSS-Attribute werden auf einem Beispieltext verwendet:

* font-size
* font-family
* font-weight
* text-align
* line-height
* hyphens
* color
* text-decoration
